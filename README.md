# 🖐️ Hand Gesture Trainer
A browser-based AI tool to train custom hand gesture models in real-time using **TensorFlow.js** and **MediaPipe**.


## 🚀 Features
- 🎥 Real-time webcam input  
- ✋ Capture custom gesture samples  
- 🧠 Train your own model in-browser  
- ⚡ Instant gesture prediction  
- 💾 Export and reuse trained data  


## 🛠️ Tech Stack
- **TensorFlow.js**
- **MediaPipe Holistic**
- **JavaScript (Vanilla)**
- HTML + CSS


## ▶️ How It Works
1. Record gesture samples (Clone / Other)
2. Train the model directly in the browser
3. Test gesture detection in real-time


## 💡 Use Case
This project demonstrates how custom machine learning models can be trained and deployed entirely in the browser without any backend.
